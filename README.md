# Mikka's Backrooms Mod

A Fabric mod for Minecraft 1.21.1 that brings the eerie and liminal world of the Backrooms to your game. Explore new dimensions, encounter terrifying entities, and try to keep your sanity intact.

## Installation

1. Download and install **Fabric Loader** for Minecraft 1.21.1.
2. Download the **Fabric API** mod and place it in your `mods` folder.
3. Download the latest release of **Mikka's Backrooms Mod** and place it in your `mods` folder.
4. Launch the game and enjoy!

## Development

To build the mod from source:

1. Clone the repository.
2. Open a terminal in the project root.
3. Run `./gradlew build` (or `gradlew build` on Windows).
4. The built jar file will be in `build/libs`.
